Flutterでモバイルアプリを開発しています。

インプリメント済みの主な機能です。
- カテゴリ一覧表示
- 板一覧表示
- スレ一覧表示
- レス一覧表示
- スライドアニメーションでの画面間の遷移
- 左から右へのフリックでの前画面への遷移
- レス一覧での画像のサムネイル表示とタップでの拡大
